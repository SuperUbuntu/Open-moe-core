# 如何给 Moe Core 做贡献

Moe Core 是一个核心引擎发行版。贡献应优先围绕核心运行时、插件宿主、消息网关、模型接入、配置、国际化、记忆服务和文档维护展开。

## 范围

- 可以提交核心服务 bug 修复、文档修正、测试补充和适配器中立的改进。
- 不在本仓库新增平台专属部署教程、社区入口或捆绑式平台适配器。
- QQ、NapCat、OneBot 等平台适配器应在外部仓库维护，并通过插件运行时或消息网关接入。
- 修改 `src/A_memorix` 前必须先阅读 `src/A_memorix/MODIFICATION_POLICY.md`。

## 提交建议

提交 Issue 或 PR 时请说明：

- 变更目的和影响范围
- 是否影响插件/runtime API
- 是否需要数据迁移或配置迁移
- 已执行的测试或无法执行测试的原因

## 法律声明

贡献者必须确认自己有权提交相关代码或文档。提交内容将按本项目的 GPL-3.0 许可发布。

Moe Core 派生自 MaiBot，相关来源说明见仓库根目录的 `NOTICE`。
